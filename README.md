# Alex Rivera Portfolio

Personal portfolio website for Alex Rivera — full-stack / frontend-focused web developer.

Dark refined aesthetic with charcoal base and warm amber/copper accents.

## Stack

- React 19 + TypeScript + Vite
- TanStack Start / Router
- Tailwind CSS v4
- Lucide icons + Sonner toasts

## Run

```bash
npm install
npm run dev
```

Listens on `0.0.0.0:8080`.
